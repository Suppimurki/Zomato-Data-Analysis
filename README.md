# Zomato-Data-Analysis
Analyzed Zomato restaurant records using SQL, Excel, and Power BI to generate performance insights. Built interactive dashboards with DAX and drill through to track ratings, service trends across 15 plus cities. Improved data accuracy by 25 percent and reduced reporting time by 30 percent through structured data modelling and automation.

Excel
SQL
